NAME Danh Phan SYNOPSIS danh [OPTION ...] jason [OPTION ...] OPTIONS -h, --help Prints a short help message and exits. -v, --version Prints version information and exits. -e, --email=ADDRESS Specifies the email address of the site owner. The default is input@danh.computer. -g, --github=USERNAME Specifies the GitHub username of the site owner. The default is ibokuri. -E, --employer=EMPLOYER Specifies the current employer of the site owner. The default is DigitalOcean. SEE ALSO 2024‐03‐02  Test Blog Post
